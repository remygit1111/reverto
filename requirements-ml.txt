# ML dependencies — not required for paper / live trading.
# Install with: pip install -r requirements-ml.txt
#
# Audit v26-26: ``-c requirements.txt`` pins every shared transitive
# dependency (numpy, pandas, scipy, etc.) to the same versions as
# the core install. Without this constraint, pip can resolve a newer
# numpy during an ML install, which causes runtime incompatibilities
# when the ML subsystem runs in the same process as the paper engine.
-c requirements.txt
